Create /devweb and deploy content

  1. Create directory /devweb and set ownership to group devops.
  2. Set SELinux context type for /devweb to an httpd type.
  3. Set permissions: user=rwx, group=rwx, others=rx and set group special permission (setgid) on /devweb.
  4. Create /devweb/index.html containing the text "Developement".
  5. Link /devweb to /var/www/html/devweb.
